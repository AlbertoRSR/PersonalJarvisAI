# PersonalJarvisAI
An AI for your personal computer that is able to control and do whatever you ask him. The projects is made in Spanish, my native language

This is a personal project. So for now on, im going to write all in Spanish.

La idea es crearme un agente personal que tenga el control completo de mi ordenador y ejecute tareas con comandos de voz. Basicamente una especie de Jarvis de la pelicula Iron Man.

Ire subiendo las actualizaciones y como voy acanzando en el projecto

## Funcionamiento actual
- Reconocimiento de voz mediante Faster-Whisper
- Integración con modelos LLM locales mediante Ollama.
- Sistema de comandos de voz.
- Apertura de aplicaciones mediante comandos naturales.
- Limpieza y normalización automática del texto reconocido.
- Gestión automática de archivos temporales.

## Tecnologías utilizadas

- Python
- Ollama
- Qwen3
- Faster-Whisper
- SoundDevice
